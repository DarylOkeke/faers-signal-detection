# Results

This directory stores outputs generated by the analysis pipeline.

## Figures

- `figures/cardiac_combined_plot.png`: Combined bar chart and PRR forest plot for the cardiac-endpoint analysis across all cohorts.

## Tables

- `tables/cardiac_complete.csv`: Full signal metrics for all cohort × reaction PT combinations (output of `pipeline/02_compute_summary.py`).
- `tables/minoxidil_trimmed.csv`: Trimmed decision table for the four pre-specified cardiac endpoints across Minoxidil and Hydralazine cohorts (output of `pipeline/03_make_trimmed_tables.py`).
- `tables/method_controls.csv`: Positive and negative control validation results (output of `tests/test_controls.py` when run as a script).

Regenerate all outputs with:

```bash
make pipeline
make analysis
```
